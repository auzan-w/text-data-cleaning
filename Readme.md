# Text Cleaning App
App to clean raw text or `.csv` file as input.


1. Run `app.py`
2. Copy and paste the URL to browser

## API endpoint
Text input API:
``sh
http://127.0.0.1:5000/api/input
``
Key: `text_input`
Value: [your text input]

Text input API:
``sh
http://127.0.0.1:5000/api/upload
``
Key: `file`
Value: [upload your .csv file]
